Detailed Description

负责文件列表组件的初始化及状态管理

负责文件列表浮动窗口组件的界面初始化， 以及后续的文件列表的管理。 与文件相关的动作比如打开文件、打开文件夹等均由MainWindow进行消息传递转发到此处进行处理。
